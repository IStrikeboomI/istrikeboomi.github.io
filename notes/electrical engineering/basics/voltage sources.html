We've discussed voltage sources earlier, but what really is a voltage source?

A voltage source is a two-terminal device that maintains a fixed voltage

This is irregardless of the current being drawn from it

Examples of common voltage sources are batteries, wall power, solar cells, USB power

Using KVL, voltage sources in series add voltage

Using KCL, voltage sources in parallel maintain the same voltage, but it increases max current you can draw

Real vs Ideal Voltage Sources

There are 2 kinds of voltage sources, real and ideal

An ideal voltage source maintains a constant voltage regardless of the current drawn

While a real voltage source has some internal resistance that causes the voltage to drop as current increases.

This internal resistance is usually very small, but it is still there

For example, a 9V battery may only provide 8.5V when a high current is drawn from it

This also explains with using ohm's law why we don't get infinite current

For an ideal voltage source $5 V = 0\Omega \times I$ has no solution

But for a real voltage source $5 V = 0.1\Omega \times I$ has a solution of $I = 50 A$

Current Limited Voltage Sources

Some voltage sources are current limited

This means that they can only provide a max current

For example, a USB power source can only provide $200$mA of current

The reason why the sources are current limited is because too much current leads to heat

If you want more current, you either need a different power source or multiple sources in parallel

Types of Voltage

There are 2 main types of voltage sources, AC and DC

AC stands for alternating current, while DC stands for direct current

AC voltage sources change polarity periodically (sinusoidal), while DC voltage sources maintain a constant value

Most household power is AC (120 V), while batteries are DC (1.5V, 3V, 5V, 9V)

We will cover circuits in AC and DC later in more detail, just know they're not interchangeable

If we graph DC voltage over time, it would look like a straight horizontal line

If we graph AC voltage over time, it would look like a sine wave

Now this for an ideal source, if we graph a real source it would look a little more noisy
