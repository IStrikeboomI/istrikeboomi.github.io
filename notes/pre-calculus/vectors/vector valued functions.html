A vector valued function $\vec{r}(t)$ is

$$\begin{align} \vec{r}(t) = \left< f(t),g(t),h(t) \right> \end{align}$$

Where $f(t),g(t),h(t)$ are real functions, and $\vec{r}(t)$ outputs a vector

Why Do We Need Vector Valued Functions?

What you might notice are the indivual components $f, g, h$, which all share a varaible

This appears a lot like a parametric function, which it can be represented as

Vector valued functions let us represent parametric equations in one function

Just like parametric functions, you can graph them the same way

When graphing vector valued functions, you mark the line that the end makes as $t$ increases

The following vector valued function and parametric equation look the same on the graph

$$\begin{align} \vec{r}(t) &= \left< f(t),g(t),h(t) \right>\\\\ x &= f(t) \\ y &= g(t) \\ z &= h(t) \end{align}$$

The properties of vector valued functions are the same as regular vectors


Below is the graph of $ \vec{r}(t) = \left< \cos(t),\sin(t),t \right>$, this particular shape is called a helix

The green vector is the vector the function output, the red line is the curve it forms

Domain

The domain of a vector valued function is the intersection of the domain of each component

This is because in order for a value to be in the domain, it has to exist for all the components

For example the following vector valued function:

$$\begin{align} \vec{r}(t) = \left< \ln(t),t^3,\frac{1}{t - 2} \right> \end{align}$$

The domain would be $(0,2) \ \bigcup \ (2,\infty)$

This is because $ln(x)$ has a domain of $(0,\infty)$, $t^3$ has $\mathbb{R}$, and $1/t-2$ has $(-\infty,2) \ \bigcup \ (2, \infty)$

Practice Problems
  • Find the domain of $\vec{r}(t) = \left< \sqrt{4 - t^2},\tan(t),4t \right>$

    $$\begin{align} \vec{r}(t) = \left< \sqrt{4 - t^2},\tan(t),4t \right> \end{align}$$

    The domain of $\sqrt{4 - t^2}$ is $[-2,2]$

    The domain of $tan(t)$ is $t \neq \frac{\pi}{2} + n\pi$ where $n \in \mathbb{Z}$

    The domain of $4t$ is $\mathbb{R}$

    The intersection of these domains is $[-2,-\pi/2) \ \bigcup \ (-\pi/2,\pi/2) \ \bigcup \ (\pi/2,2]$

  • Find a vector valued function that represents of the curve of the intersection of $z = \sqrt{x^3 - y^2}$ and $x+y=1$

    $$\begin{align} z = \sqrt{x^3 - y^2} \\ x+y=1 \end{align}$$

    The way we approach this problem is to make a parametric equation that we will turn into a vector at the end

    We can turn this into a parametric by setting a variable to $t$ and solving the other 2

    Here I am going to take $x = t$

    $$\begin{align} x &= t \\ t+y=1 &\rightarrow y = 1-t \end{align}$$

    We can plug this into $z$

    $$\begin{align} z &= \sqrt{t^3 - (1 - t)^2} \\ &= \sqrt{t^3 - (1 - 2t + t^2)} \\ &= \sqrt{t^3 - 1 + 2t - t^2)} \\ &= \sqrt{t^3 - t^2 + 2t - 1} \\ \end{align}$$

    So we have our parametrics, we can turn this into a vector

    $$\begin{align} \vec{r}(t) = \left< t,1-t,\sqrt{t^3 - t^2 + 2t - 1} \right> \end{align}$$

    If we graph the two 3d functions above and then the vector, we can change values of $t$ and see it trace the curve

  • Represent the line segment that contains the points $P(1,-2,3) Q(-2,3,1)$ as a vector valued function

    $$\begin{align} P(1,-2,3) &\ Q(-2,3,1) \\ \vec{PQ} &= \left< -3, 5,-2 \right> \\ \end{align}$$

    Using $P$ as the refernce point (Q works too)

    $$\begin{align} x &= 1 - 3t \\ y &= -2 + 5t \\ z &= 3 - 2t \\ \vec{r} &= \left< 1-3t,-2+5t,3-2t \right> \end{align}$$
