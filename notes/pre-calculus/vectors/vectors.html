Vectors are mathematical objects that have a direction and magnitude (length)

Vectors can be thought of as a directed line segment that can be in 2,3,4, any number of dimensions

One way a vector is written is a bold character with an arrow on top

$$\begin{align} \vec{v} = \left< v_1,v_2,v_3,...,v_n \right> \end{align}$$

Here is a graph of the vector $\vec{v} = \left< 2,3 \right>$

When you graph vectors you start off at the origin and create and arrow

A position vector is a vector made from 2 points

If there is a point $P$ and another point $Q$, the position vector is written as

$$\begin{align} \vec{PQ} = \left< Q_1 - P_1,Q_2 - P_2,Q_3 - P_3,...,Q_n - P_n \right> \end{align}$$

It's also important to note that the direction of $\vec{PQ}$ matters, what this is means is

$$\begin{align} \vec{PQ} \neq \vec{QP} \end{align}$$

What the opposite vector will look like is the same line, but pointing in the other direction


The magniute of the vector is the length or distance of the vector, it can be found by using the distance formula

$$\begin{align} ||\vec{v}|| &= \sqrt{v_1^2 + v_2^2 + v_3^2 + ... + v_n^2} \\ &= \sqrt{(Q_1 - P_1)^2 + (Q_2 - P_2)^2 + (Q_3 - P_3)^2 + ... + (Q_n - P_n)^2} \\ \end{align}$$

For example the magnitude of the previous vector in the graph is:

$$\begin{align} ||\vec{v}|| &= \sqrt{2^2 + 3^2} = \sqrt{12} \\ \end{align}$$

Basic Operations And Properties

$\vec{u}, \vec{v}, \vec{w}$ are vectors and $c$ and $d$ is a constant

Sum: $\vec{u} + \vec{v} = \left< u_1 + v_1, u_2 + v_2, u_3 + v_3, ..., u_n + v_n \right>$

Scalar: $c\vec{u} = \left< cu_1, cu_2, cu_3, ..., cu_n \right>$

Negation: $-\vec{u} = \left< -u_1, -u_2, -u_3, ..., -u_n \right>$

Difference: $\vec{u} - \vec{v}= \left< u_1 - v_1, u_2 - v_2, u_3 - v_3, ..., u_n - v_n \right>$

The reason vector addition works like this is because the first vector offsets and the second vectors offsets from that offset

Commutative Property: $\vec{u} + \vec{v} = \vec{v} + \vec{u}$

Associative Property: $\vec{u} + (\vec{w} + \vec{v}) = (\vec{u} + \vec{w}) + \vec{v}$

Scalar Associative Property: $c(d\vec{u}) = d(c\vec{u})$

Scalar Distributive Property: $(c + d)\vec{u} = c\vec{u} + d\vec{u}$

Alternate Notation

Another way of writing vectors is to write it as a sum of indivual component vectors

$$\begin{align} \textbf{i} &= \left< 1,0,0\right> \\ \textbf{j} &= \left< 0,1,0\right> \\ \textbf{k} &= \left< 0,0,1\right> \\ \end{align}$$

For example: the vector $\vec{v} = \left< 5,3,2\right>$ would be written as:

$$\begin{align} \vec{v} = 5\textbf{i} + 3\textbf{j} + 2\textbf{k} \end{align}$$

This is usually called ijk notation and it is useful for noting direction


There is also column notation where you represent the components of a vector as a 1 column matrix

The above vector can be written like so

$$\begin{align} \begin{bmatrix} 5 \\ 3 \\ 2 \end{bmatrix} \end{align}$$

There is also similarly row notation, where instead its represented in a matrix with 1 row

$$\begin{align} \begin{bmatrix} 5 & 3 & 2 \end{bmatrix} \end{align}$$

Unit Vectors

A vector with a magnitude of $1$ is called a unit vector and represented with a "hat" (carat) on top

A vector with a magnitude of $0$ is called a null vector and is represented by $\vec{0}$

Unit vectors are denoted by a $\hat{u}$

A unit vector can be found with the following formula

$$\begin{align} \hat{v} = \frac{\vec{v}}{||\vec{v}||} \end{align}$$

What the unit vector represents is just the direction of the vector without any distancec

If we graph the unit vector on top of the original vector it will be pointing the same way, just scaled differntly

That's what dividing by the magnitude does, it scales it

Let's try to find the unit vector and magnitude for $\vec{v} = \left< 2,12,8 \right>$

$$\begin{align} ||\vec{v}|| = \sqrt{2^2 + 12^2 + 8^2} = \sqrt{212} = 2\sqrt{53} \\ \hat{v} = \frac{\vec{v}}{||\vec{v}||} = \frac{1}{2\sqrt{53}} \left< 2,12,8 \right> = \left< \frac{1}{\sqrt{53}},\frac{6}{\sqrt{53}},\frac{4}{\sqrt{53}} \right> \end{align}$$

You could further rationalize the denominator, but this is good enough

Practice Problems
  • Solve $2\left< 1,0,4 \right> + \left< 4,7,22 \right> - \left< 5,2,2 \right>$

    $$\begin{align} 2\left< 1,0,4 \right> + \left< 4,7,22 \right> - \left< 5,2,2 \right> &= \left< 2,0,8 \right> + \left< 4,7,22 \right> - \left< 5,2,2 \right> \\ &= \left< 2 + 4 - 5,0 + 7 - 2,8 + 22 - 2 \right> \\ &= \left< 1,5,28 \right> \\ \end{align}$$

  • Find the unit vector of $\left< 5,12,22,3 \right>$

    $$\begin{align} \vec{v} &= \left< 5,12,22,3 \right> \\ ||\vec{v}|| &= \sqrt{5^2 + 12^2 + 22^2 + 3^2} = \sqrt{662} \\ \hat{v} &= \frac{\vec{v}}{\sqrt{662}} = \left< \frac{5}{\sqrt{662}},\frac{12}{\sqrt{662}},\frac{22}{\sqrt{662}},3\sqrt{662} \right> \end{align}$$

  • Solve $0\left< 2,7,36,124,26,361,23,61,3 \right>$

    $$\begin{align} 0\left< 2,7,36,124,26,361,23,61,3 \right> = \left< 0,0,0,0,0,0,0,0,0 \right> \end{align}$$

  • Find the Unit Vector of the Position Vector of the Points: $P = (2,3), Q=(-4,2)$

    $$\begin{align} \vec{PQ} &= \left<-4 - 2, 2 - 3 \right> = \left<-6, -1 \right> \\ ||\vec{PQ}|| &= \sqrt{(-6)^2 + (-1)^2} = \sqrt{37} \\ \hat{PQ} &= \frac{\vec{PQ}}{\sqrt{37}} = \left<\frac{-6}{\sqrt{37}}, \frac{-1}{\sqrt{37}} \right> \end{align}$$
