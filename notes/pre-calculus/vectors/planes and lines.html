Vectors make 3D graphing easier, first we need to establish some vocabulary

If you want to make 3D graphs, I like to use Desmos

Planes

A plane can be thought of as the 3D extension of a 2D line

It is a flat 2D surface that extends indefinitely

Below is the plane $2x + y - 2z - 3 = 0$, with the normal vector shown at different angles

A plane containing the point $P(x_1,y_1,z_1)$, with a normal vector of $\vec{n} = \left< a,b,c \right>$ is:

$$\begin{align} a(x-x_1) + b(y-y_1) + z(z-z_1) = 0 \end{align}$$

This can be written more generally (where $d = -ax_1 -by_1 -cz_1$),

$$\begin{align} ax + by + cz + d = 0 \end{align}$$

The normal vector $\vec{n}$ of a plane is a vector that is orthogonal to the plane

The reason why planes are defined using the normal vector can be proven by the following

First define $Q(x,y,z)$ as a point on the plane, and make the vector $\vec{PQ}$

$$\begin{align} \vec{PQ} = \left< x-x_1,y-y_1,z-z_1 \right> \end{align}$$

Since $\vec{PQ}$ lies on the plane (because both points are on the plane), and $\vec{n}$ is orthogonal, that means the Dot Product is zero

$$\begin{align} \vec{n} \cdot \vec{PQ} = 0 \rightarrow a(x-x_1) + b(y-y_1) + c(z-z_1) = 0 \end{align}$$

To find the angle between two planes, the angle between the planes would reflect throughout the whole plane

This includes the normal vectors

$$\begin{align} \cos(\theta) = \frac{|\vec{n_1} \cdot \vec{n_2} |}{||\vec{n_1}||\cdot||\vec{n_2}||} \end{align}$$

The distance between a point $Q(x_1,y_1,z_1)$ and a plane $ax + by + cz + d = 0$ is:

$$\begin{align} D = \frac{|ax_1 + by_1 + cz_1 + d|}{\sqrt{a^2 + b^2 + c^2}} \end{align}$$

The way we prove is this is to first define a point on the plane $P(x_2,y_2,z_2)$

We need to find the distance from $Q$ to the plane, we can map the distance onto the normal vector using a projection onto n

$$\begin{align} \vec{PQ} &= \left< x_2 - x_1, y_2 - y_1, z_2 - z_1 \right> \\ D = |proj_{\vec{n}}{\vec{PQ}}| &= \frac{|\vec{PQ} \cdot \vec{n}|}{||\vec{n}||} = \frac{|ax_1 + by_1 + cz_1 + d|}{\sqrt{a^2 + b^2 + c^2}} \end{align}$$

Lines

Traditional lines also exist in the 3d plane, but they have to be written parametrically

The equation of a line parallel to $\vec{v} = \left< a,b,c\right>$ and passing through the point $P(x_1,y_1,z_1)$ is:

$$\begin{align} x &= x_1 + at \\ y &= y_1 + bt \\ z &= z_1 + ct \\ \end{align}$$

The line can also be written in symmetric form,

$$\begin{align} \frac{x - x_1}{a} = \frac{y - y_1}{b} = \frac{z -z_1}{c} \end{align}$$

Below is the line $x = 1 + t, y = 2 - \frac{1}{2}t, 1 + t$ at different angles

The distance between a point $Q$ and a point on the line $P$ and a direction vector $\vec{u}$ is

$$\begin{align} D = \frac{||\vec{PQ} \times \vec{u}||}{||\vec{u}||} \end{align}$$

The direction vector can be a vector with just the compoennts as the parametric parts of the line

This can be proved by the fact that a right triangle is formed between $P$ and $Q$

$$\begin{align} D = ||\vec{PQ}||\sin(\theta) = \frac{||\vec{PQ}||\cdot ||\vec{u}|| \cdot \sin(\theta)}{||\vec{u}|| } = \frac{||\vec{PQ} \times \vec{u}||}{||\vec{u}||} \end{align}$$
