Like the regular trigonometric functions, which take in an angle and output a ratio

The inverse trigonometric functions take in a ratio and output an angle

The inverse trigonometric functions are:

$$\begin{align} \sin^{-1}\left(\frac{\text{opposite}}{\text{hypotenuse}}\right) = \theta &\ \cos^{-1}\left(\frac{\text{adjacent}}{\text{hypotenuse}}\right) = \theta \\ \csc^{-1}\left(\frac{\text{hypotenuse}}{\text{opposite}}\right) = \theta &\ \sec^{-1}\left(\frac{\text{hypotenuse}}{\text{adjacent}}\right) = \theta \\ \tan^{-1}\left(\frac{\text{opposite}}{\text{adjacent}} \right) = \theta &\ \cot^{-1}\left(\frac{\text{adjacent}}{\text{opposite}} \right) = \theta \\ \end{align}$$

There is some ambiguity to whether the $-1$ is an exponent or denoting the inverse

To remedy this, there is an alternate notation where you put arc- in front of the function

$$\begin{align} \arcsin\left(\frac{\text{opposite}}{\text{hypotenuse}}\right) = \theta &\ \arccos\left(\frac{\text{adjacent}}{\text{hypotenuse}}\right) = \theta \\ arccsc\left(\frac{\text{hypotenuse}}{\text{opposite}}\right) = \theta &\ arcsec\left(\frac{\text{hypotenuse}}{\text{adjacent}}\right) = \theta \\ \arctan\left(\frac{\text{opposite}}{\text{adjacent}} \right) = \theta &\ arccot\left(\frac{\text{adjacent}}{\text{opposite}} \right) = \theta \\ \end{align}$$

Both of these notations are widely accepeted and represent the same thing

A lot of calculators might not have $arccsc,arcsec,arccot$

If you want to use these functions, you can use plug $1/x$ into $arcsin,arccos,arctan$ where $x$ is your input
