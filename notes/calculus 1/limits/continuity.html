Imagine a function $f(x)$ is a continuous function

What this means intuitively is that there are no breaks or sudden changes in its output (y value)

This means that any holes and asymptotes makes the function not continuous

Another way to think of it is if can you draw the function with a pencil without needing to lift it up, it's continous

Observe the graph of $f(x) = x^2$ below

The graph has no breaks or sudden or jumps, which continues throughout the function, meaning it is continous

Now observe the graph of $f(x) = sec^{-1}(x)$

There is a sudden jump between $-1$ and $1$ showing that this is not continous

A function that is NOT continuous is called discontinuous

For a more precise definition, a function $f(x)$ is continuous at $x=c$, if and only if

$$\begin{align} f(c) \text{ exists} \tag{1} \\ \lim_{n \to c}{f(x)} \text{ exists} \tag{2} \\ \lim_{n \to c}{f(x)} = f(c) \tag{3} \end{align}$$

Criteria (1), (2), and (3) all have to satisified

If Criteria (1) and (3) are failed, then it is referd to as a removable discontinuity (This is what occurs with a hole)

If Criteria (2) and (3) are failed, then it is referd to as a non-removable discontinuity or also jump discontinuity

If all criteria fail, then it is referd to as an infinite discontinuity if either side of the limit goes to infinity
