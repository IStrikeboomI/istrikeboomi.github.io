The Squeeze Theorem States:

For all $x$ on $[a,b]$ (except possibly $c$)

$$\begin{align} h(x) \leq f(x) \leq g(x) \end{align}$$

And

$$\begin{align} \lim_{x \to c} h(x) &= L \\ \lim_{x \to c} g(x) &= L \\ \end{align}$$

Then

$$\begin{align} \lim_{x \to c} f(x) &= L \\ \end{align}$$

What Does That Mean?

What the squeeze theorem says is that if you have function $h(x), f(x), g(x)$ all defined on an interval

And $f(x)$ is between $h(x)$ and $g(x)$

The limit for $f(x)$ at $c$ is the same if the limits at $c$ for $h(x)$ and $g(x)$ are equal

Look at the following graph for an example

In the above graph, $f(x)$ is between $h(x)$ and $g(x)$

As $x \to 0$, all three functions approach to $0$

Lets solve an example

$$\begin{align} \lim_{x \to 0} x^2sin\left(\frac{1}{x}\right) \end{align}$$

If we use direct substitution, we get $sin(\infty)$, which doesn't exist

We can't really use any properties to simplyfy this, but we can apply the squeeze theorem

First, we need to find two functions that $x^2sin\left(\frac{1}{x}\right)$ lies between

We can first start by using the range of $sin(x)$, which lies between $[-1, 1]$

$$\begin{align} -1 \leq &sin\left(x\right) \leq 1 \\ -1 \leq &sin\left(\frac{1}{x}\right)\leq 1 &&\text{$sin\left(\frac{1}{x}\right)$ has the same range}\\ -x^2 \leq x^2&sin\left(\frac{1}{x}\right)\leq x^2 &&\text{Multiply all terms by $x^2$}\\ \lim_{x \to 0}-x^2 \leq \lim_{x \to 0}x^2&sin\left(\frac{1}{x}\right) \leq \lim_{x \to 0}x^2 &&\text{Apply Squeeze Theorem by taking limit}\\ 0 \leq \lim_{x \to 0}x^2&sin\left(\frac{1}{x}\right) \leq 0\\ \lim_{x \to 0}x^2&sin\left(\frac{1}{x}\right) = 0\\ \end{align}$$

Practice Problems
  • $\lim_\limits{x \to 0}xe^{cos(1/x)}$

    $$\begin{align} \lim_\limits{x \to 0}xe^{cos(1/x)} \end{align}$$

    $$\begin{align} -1 \leq &cos\left(\frac{1}{x}\right) \leq 1 \\ e^{-1} \leq &e^{cos\left(1/x\right)} \leq e^{1} \\ xe^{-1} \leq &xe^{cos\left(1/x\right)} \leq xe^{1} \\ \lim_{x \to 0}xe^{-1} \leq &\lim_{x \to 0}xe^{cos\left(1/x\right)} \leq \lim_{x \to 0}xe^{1} \\ 0 \leq &\lim_{x \to 0}xe^{cos\left(1/x\right)} \leq 0 \\ & \lim_{x \to 0}xe^{cos\left(1/x\right)} = 0 \end{align}$$

  • Find $\lim_\limits{x \to 2}f(x)$, if $-x^2 + 4x - 3 \leq f(x) \leq x^2 - 4x + 5$

    $$\begin{align} \lim_\limits{x \to 2} -x^2 + 4x - 3 &= 1 \\ \lim_\limits{x \to 2} x^2 - 4x + 5 &= 1 \\ \end{align}$$

    Since $f(x)$ is between, we can use squeeze theorem

    $$\begin{align} \lim_\limits{x \to 2} f(x) &= 1 \\ \end{align}$$

Proof

Given:

$$\begin{align} h(x) \leq f(x) \leq g(x) \end{align}$$

$$\begin{align} \lim_\limits{x \to c} h(x) &= L \\ \lim_\limits{x \to c} g(x) &= L \\ \end{align}$$

By formal definition of limit:

We need to find a $\delta > 0$ given $\epsilon > 0$ such that

$$\begin{align} |f(x) - L| < \epsilon \\ 0 < |x - c| < \delta \end{align}$$

And since we are given the limits for $h(x)$ and $g(x)$ exist

Then there exists some $\delta_1 > 0$ such that

$$\begin{align} |h(x) - L| < \epsilon \text{ for all } 0 < |x - c| < \delta_1 \end{align}$$

We can write rewrite the first part

$$\begin{align} &|h(x) - L| < \epsilon \\ -\epsilon < &h(x) - L < \epsilon\\ L - \epsilon < &h(x) < \epsilon + L \\ \end{align}$$

To get

$$\begin{align} L - \epsilon < &h(x) < \epsilon + L \text{ for all } 0 < |x - c| < \delta_1 \end{align}$$

We can also apply this to $g(x)$ for a $\delta_2 > 0$

$$\begin{align} L - \epsilon < &g(x) < \epsilon + L \text{ for all } 0 < |x - c| < \delta_2 \end{align}$$

And since $h(x) \leq g(x)$

We can take $\delta = min\{\delta_1,\delta_2\}$ to get

$L - \epsilon < h(x) \leq g(x) < \epsilon + L \text{ for all } 0 < |x - c| < \delta$

But since, $h(x) \leq f(x) \leq g(x)$, we can replace with $f(x)$

$L - \epsilon < f(x) < \epsilon + L \text{ for all } 0 < |x - c| < \delta$

Which Leads to

$$\begin{align} \lim_\limits{x \to c} f(x) &= L \\ \end{align}$$
