The following limit is very important:

$$\begin{align} \lim_{x \to 0}\frac{sin(x)}{x} = 1 &\tag{1} \end{align}$$

This limit shows up in the derivative of sin (which we will talk about in a later note)

If we try direct substitution, we get $\frac{0}{0}$, so we need to do something else

We need to use geometry and the squeeze theorem

There is also a corollary

$$\begin{align} \lim_{x \to 0}\frac{cos(x) - 1}{x} = 0 \end{align}$$

Proof

First, create a unit circle (circle with radius of 1)

Next create an arc sector with an angle $\theta$

Using the arc sector length formula, $\theta \cdot r$, we get an arc length of $\theta$

Lets make 2 right triangles

One triangle is inside the circle with a hypotenuse of 1, we mark this triangle green

The other triangle has it's adjacent length of 1, this places part of the triangle outside the circle, we mark this triangle blue

We can find the opposite length for both of these triangles

$$\begin{align} sin(\theta) = \frac{opposite}{hypotenuse} \rightarrow sin(\theta) = \frac{opposite}{1} \rightarrow sin(\theta) = opposite \end{align}$$

$$\begin{align} tan(\theta) = \frac{opposite}{adjacent} \rightarrow tan(\theta) = \frac{opposite}{1} \rightarrow tan(\theta) = opposite \end{align}$$

Looking at the picture, we can see that the order of the length sizes are $sin(\theta)$, then $\theta$, then $tan(\theta)$

Since our limit variable x is approaching 0, it means if we plug 0 into all 3 $\theta$, we get 0 for all (making them equal)

This can be expressed as the inequality:

$$\begin{align} sin(\theta) \leq \theta \leq tan(\theta) \\ sin(\theta) \leq \theta \leq \frac{sin(\theta)}{cos(\theta)} &&\text{Rewrite tan as $\frac{sin}{cos}$}\\ \frac{1}{sin(\theta)} \geq \frac{1}{\theta} \geq \frac{cos(\theta)}{sin(\theta)} &&\text{Take Reciprocal (Don't forget to switch inequality signs)}\\ 1 \geq \frac{sin(\theta)}{\theta} \geq cos(\theta)&&\text{Multiply term each by $sin(\theta)$}\\ \end{align}$$

We can apply squeeze theorem here

$$\begin{align} \lim_{\theta \to 0}1 \geq &\lim_{\theta \to 0}\frac{sin(\theta)}{\theta} \geq \lim_{\theta \to 0}cos(\theta) \\ 1 \geq &\lim_{\theta \to 0}\frac{sin(\theta)}{\theta} \geq 1 \\ \end{align}$$

This means that:

$$\begin{align} \lim_{\theta \to 0}\frac{sin(\theta)}{\theta} = 1 \end{align}$$

Corrollary

This is to prove the following (also important) limit:

$$\begin{align} \lim_{x \to 0}\frac{cos(x) - 1}{x} = 0 \end{align}$$

We can use some algebra to manipulate the above limit so we can use (1), and then direct substituion

$$\begin{align} \lim_{x \to 0}\frac{cos(x) - 1}{x} &= \lim_{x \to 0}\frac{cos(x) - 1}{x} \cdot \frac{cos(x) + 1}{cos(x) + 1} &&\text{Multiply By Conjugate} \\ &= \lim_{x \to 0}\frac{cos(x)^2 - 1^2}{x(cos(x) + 1)} &&\text{Using Difference of Two Squares on Numerator} \\ &= \lim_{x \to 0}\frac{-(-cos(x)^2 + 1)}{x(cos(x) + 1)} &&\text{Factor Out Negative Sign} \\ &= \lim_{x \to 0} - \frac{1 - cos(x)^2}{x(cos(x) + 1)} \\ &= \lim_{x \to 0} - \frac{sin^2(x)}{x(cos(x) + 1)} &&\text{Using Rearranged version of $sin^2(x) + cos^2(x) = 1$} \\ &= \lim_{x \to 0} - \frac{sin(x)}{x} \cdot \frac{sin(x)}{cos(x) + 1} &&\text{Split fraction}\\ &= \lim_{x \to 0} - \frac{sin(x)}{x} \cdot \lim_{x \to 0}\frac{sin(x)}{cos(x) + 1} &&\text{Using limit product property} \\ &= -1 \cdot \lim_{x \to 0}\frac{sin(x)}{cos(x) + 1} &&\text{Using (1)} \\ &= -1 \cdot \frac{sin(0)}{cos(0) + 1} \\ &= -1 \cdot \frac{0}{1 + 1} \\ &= -1 \cdot 0 \\ &= 0 \end{align}$$
