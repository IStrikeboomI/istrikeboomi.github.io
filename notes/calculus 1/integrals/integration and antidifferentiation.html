Anti-Differentiation is the inverse (opposite) of differentiation

We usually denote the anti-differentiation of $f(x)$ as $F(x)$

Taking the derivative of the anti-derivative returns back the original function

$$\begin{align} F'(x) = f(x) \end{align}$$

The general form of anti-differentiation where all the possible anti-derivatives exist is called an Indefinite Integral, or just an Integral

The way an integral is written is by using following

  1. The Integral Symbol $\int$

  2. The function inside (called the Integrand)

  3. A $dx$ ($x$ is the variable we working with), the $dx$ also denotes where the integral ends, think of it as parentheses

The result of an integral is the anti-derivative and a plus C ($+ C$)

$$\begin{align} \int{f(x)\ dx} = F(x) + C \end{align}$$

Or alternatively,

$$\begin{align} \int{f'(x)\ dx} = f(x) + C \end{align}$$

The plus C is referred to as the Constant Of Integration, it is mandatory to add it at the end of your indefinite integrals

If you want to know why it is mandatory and what it means, check out Constant Of Integration ($+C$)

Examples

$$\begin{align} \int{2x \ dx} \end{align}$$

What this integral means is what can we take the derivative of to get $f'(x) = 2x$

You might notice the derivative of $x^2$ is $2x$, so we can use this as the answer

$$\begin{align} \int{2x \ dx} = x^2 + C \end{align}$$

$$\begin{align} \int{cos(t) \ dt} \end{align}$$

What this integral means is what can we take the derivative of to get $f'(t) = cos(t)$

You might notice the derivative of $sin(t)$ is $cos(t)$, so we can use this as the answer

$$\begin{align} \int{cos(t) \ dt} = sin(t) + C \end{align}$$

Practice Problems
  • $\int{ 3x^2 \ dx}$

    $$\begin{align} f'(x) &= 3x^2 \\ f(x) &= x^3 \\ \int{ 3x^2 \ dx} &= x^3 + C \\ \end{align}$$

  • $\int{ sec^2(v) \ dv}$

    $$\begin{align} f'(v) &= sec^2(v) \\ f(v) &= tan(v) \\ \int{ sec^2(v) \ dv} &= tan(v) + C \\ \end{align}$$

  • $\int{ x \ dx}$

    $$\begin{align} f'(x) &= x \\ f(x) &= \frac{1}{2}x^2 \\ \int{ x \ dx} &= \frac{x^2}{2} + C \\ \end{align}$$
