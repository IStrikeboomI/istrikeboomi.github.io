Integral Properties are necessary to solve integrals, both definite and indefinite

A lot of the basic integral properties are just the derivative properties, but opposite

Sum/Difference Property:

$$\begin{align} \int{f(x) \pm g(x)\ dx} &= \int{f(x)\ dx} \pm \int{g(x)\ dx} \\ &\text{or for definite integrals,} \\ \int_a^b{f(x) \pm g(x)\ dx} &= \int_a^b{f(x)\ dx} \pm \int_a^b{g(x)\ dx} \end{align}$$

Constant Coefficient Property (assume $c$ is a constant):

$$\begin{align} \int{cf(x)\ dx} &= c\int{f(x)\ dx} \\ &\text{or for definite integrals,} \\ \int_a^b{cf(x)\ dx} &= c\int_a^b{f(x)\ dx} \end{align}$$

Constant Inner Coefficient Property (assume $c$ is a constant):

$$\begin{align} \int{f(cx)\ dx} &= \frac{1}{c}F(cx) + C \end{align}$$

Power Rule ($c \neq -1$):

$$\begin{align} \int{x^c \ dx} &= \frac{x^{c + 1}}{c + 1} + C \end{align}$$

Here, if $c = -1$, then the integrand is $\frac{1}{x}$, which is the derivative of $ln(x)$, in other words $\frac{d}{dx} ln(x) = \frac{1}{x}$

Negation Property:

$$\begin{align} \int_a^b{f(x)\ dx} = -\int_b^a{f(x)\ dx} \end{align}$$

This property comes about because of the way the definite integral is defined

Adding Interval Property (assume $a < b < c$):

$$\begin{align} \int_a^c{f(x)\ dx} = \int_a^b{f(x)\ dx} + \int_b^c{f(x)\ dx} \end{align}$$

This property basically means you split a definite integral's bounds

Zero Property:

$$\begin{align} \int_a^a{f(x)\ dx} = 0 \end{align}$$

There is no range or distance between $a$ and $a$, because they are the same so there is no area either

Odd Function Property (assume $f(x)$ is an odd function):

$$\begin{align} \int_{-a}^a{f(x)\ dx} = 0 \end{align}$$

Odd Functions are symmetrical about the origin, which means the left half is flipped of the right half, which both cancel out to 0

Even Function Property (assume $f(x)$ is an even function):

$$\begin{align} \int_{-a}^a{f(x)\ dx} = 2\int_{0}^a{f(x)\ dx} \end{align}$$

Even Functions are symmetrical about the y-axis, which means the left half is the mirror of the right half

This means we only have to do one half and just multiply by 2 to account for the other identical half
