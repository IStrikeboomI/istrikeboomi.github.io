Definite Integrals mainly deal with the area under a curve

The way you write a definite integral is using the following:

  1. The Integral Symbol $\int$

  2. The Lower Bound $a$

  3. The Upper Bound $b$

  4. The function inside (called the Integrand)

  5. A $dx$ ($x$ is the variable we working with), the $dx$ also denotes where the integral ends, think of it as parentheses

This results in a number that is the area under the curve of $f(x)$ between $[a,b]$ in respect to x (along the x-axis)

$$\begin{align} \int_a^b{f(x)\ dx} = A &&\text{$A$ is the area under the curve} \end{align}$$

Area is also different wheter it's above or below the x-axis

If the curve is above the x-axis, it is adding (positive) area

If the curve is below the x-axis, it is subtracting (negative) area

Example

Take a look at the following graph of $f(x) = x^{3}-7x^{2}+10x$

The light blue highlighted part between 0 and 2 is an area under the curve that we are trying to find

The way we use definite integrals is to represent the following notation:

$$\begin{align} \int_0^2{f(x)\ dx} = \frac{16}{3} \end{align}$$

Relation To Indefinite Integrals

You probably noticed that definite integrals look like indefinite integrals with the bounds, this is not a coincidence

This is by the Fundamental Theorem Of Calculus

$$\begin{align} \text{if } F(x) &= \int{f(x)\ dx} \\ \text{then } \int_a^b{f(x)} &= F(b) - F(a) \end{align}$$

Also because of this, an anti-derivative can be written as a definite integral from 0 to $x$

$$\begin{align} F(x) = \int_0^x{f(x)\ dx} \end{align}$$

A common notation usually called bar notation is to use a vertical bar with the bounds after the anti-differentiation

$$\begin{align} \int_a^b{f(x)} = \left[F(x)\right] \Big|_a^b = F(b) - F(a) \end{align}$$

IMPORTANT: What this means is that anti-differentiation is also the formula for getting the area from 0 to $x$

You might also be wondering why indefinite integrals have a plus C but not definite integrals

$$\begin{align} \int{f(x)\ dx} &= F(x) + C \\ \int_a^b{f(x)} &= (F(b) + C) - (F(a) + C) = F(b) = F(a) \end{align}$$

You get $+C$ and then a $-C$ which cancel out

Other Ways To Solve Definite Integrals

For solving (usually) polynomials, you can also use the formal definition of the definite integrals, Riemann Sums

The other most common way is to use geometry

For Example, to solve the following integral:

$$\begin{align} \int_0^3{\sqrt{9 - x^2} \ dx} \end{align}$$

It's pretty hard to find an indefinite integral, but you might notice it is the formula a semi-circle with radius 3

And since we are doing it from 0 to 3 it is only half of a semi-circle, which is a quarter of the area of a full circle

We can just use the circle area formula $\pi r^2$

$$\begin{align} \frac{\pi (3)^2}{4} &= \frac{9}{4} \pi \\ \int_0^3{\sqrt{9 - x^2} \ dx} &= \frac{9}{4} \pi \end{align}$$

Another Example, find the following definite integral using the graph

$$\begin{align} \int_{-2}^5{f(x)\ dx} \end{align}$$

The first place we can start is by seeing how we can separate the graph into different shapes

We can then add the area of all the shapes to get the total area

I identified 4 shapes to split

Let's find the areas for all of these

  1. Triangle with base of 1 and height 2 $\frac{1}{2} (1)(2) = 1$
  2. Rectangle with base of 4 and height 2 $(4)(2) = 8$
  3. Semicircle with radius 1 $\frac{1}{2} \pi (1)^2 = \pi$
  4. Rectangle with base of 3 and height 3 $(3)(3) = 9$

Adding up all the areas, $1 + 8 + 9 + \pi = 18 + \pi$, which is our answer

$$\begin{align} \int_{-2}^5{f(x)\ dx} = 18 + \pi \approx 21.14159 \end{align}$$

Practice Problems
  • $\int_0^{\pi}{sec^2(x)\ dx}$

    $$\begin{align} \int_0^{\pi}{sec^2(x)\ dx} &= \left[tan(x)\right] \biggr|_0^{\pi} \\ &= tan(\pi) - tan(0) \\ &= 0 - 0 = 0 \end{align}$$

  • $\int_2^{3}{e^x \ dx}$

    $$\begin{align} \int_2^{3}{e^x \ dx} &= \left[e^x\right] \Big|_2^3 \\ &= e^3 - e^2 = e^2(e - 1) \approx 12.6964 \end{align}$$

  • $\int_2^{5}{2 \ dx}$

    The graph of $f(x) = 2$ is just a straight line, this makes a rectangle between it and the x axis

    This means a rectangle with length 3 (from subtracting the bounds $5 - 2 = 3$) and a height of 2

    $$\begin{align} 3 \cdot 2 &= 6 \\ \int_2^{5}{2 \ dx} &= 6 \end{align}$$
