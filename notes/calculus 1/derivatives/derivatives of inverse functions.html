let $f(x)$ be a differentiable function (meaning $f'(x)$ exists)

if $f(x)$ is bijective (one-to-one) meaning each input its own unique output

Then the derivative of $f^{-1}(x)$ (the inverse of $f(x)$) can be found

Inverse Function Rule:

$$ (f^{-1})'(x) = \frac{1}{f'(f^{-1}(x))} $$

Proof

$$\begin{align} f(f^{-1}(x)) &= x \\ \frac{d}{dx}f(f^{-1}(x)) &= \frac{d}{dx}x \\ (f^{-1})'(x) \cdot f'(f^{-1}(x)) &= 1 &&\text{Using Chain Rule} \\ (f^{-1})'(x) &= \frac{1}{f'(f^{-1}(x))} \end{align}$$

Practice Problems
  • Use Inverse function rule for $\frac{d}{dx} \sqrt[3]{x}$

    We first we make &\sqrt[3]{x} - 8$ the inverse and find the original function from that

    $$\begin{align} f^{-1}(x) &= \sqrt[3]{x} \\ y &= \sqrt[3]{x} \\ x &= \sqrt[3]{y} &&\text{Switch $x$ and $y$ and solve for y}\\ (x)^{3} &= y \\ f(x) &= (x)^{3} \\ f'(x) &= 3x^2 \end{align}$$

    Now we can apply the inverse function rule

    $$\begin{align} \frac{d}{dx} \sqrt[3]{x} &= \frac{1}{3(\sqrt[3]{x^2})} \\ &= \frac{1}{3x^{\frac{2}{3}}} \end{align}$$

  • Use Inverse function rule for $\frac{d}{dx}tan^{-1}(x)$

    We first we make &tan^{-1}(x)$ the inverse and find the original function from that

    $$\begin{align} f^{-1}(x) &= tan^{-1}(x) \\ f(x) &= tan(x) \\ f'(x) &= sec^2(x) \end{align}$$

    Now we can apply the inverse function rule

    $$\begin{align} \frac{d}{dx}tan^{-1}(x) &= \frac{1}{sec^2(tan^{-1}(x))} \\ \end{align}$$

    This can be simplified further using right triangles

    The $sec$ takes in a $\theta$, and the $tan^{-1}$ outputs a $\theta$

    This means they can be used to make a triangle

    $$\begin{align} tan^{-1}(x) = \theta \rightarrow tan(\theta) = \frac{x}{1} \end{align}$$

    Since $tan(\theta) = \frac{opposite}{adjacent}$, that means we have a right triangle with opposite of $x$ and adjacent of $1$

    We can use the pythagorean theorem ($a^2 + b^2 = c^2$) to find the hypotenuse

    $$\begin{align} a^2 + b^2 &= c^2 \\ x^2 + 1^2 &= c^2 \\ c &= \sqrt{x^2 + 1} \end{align}$$
