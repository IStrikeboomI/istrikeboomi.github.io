Mean Value theorem

If a function $f(x)$ is continous on $[a,b]$ and differentiable on $(a,b)$

Then, there is a point $c$ in $(a,b)$ ($c \in (a,b)$)

$$\begin{align} f'(c) = \frac{f(b) - f(a)}{b-a} \end{align}$$

Or alternatively

$$\begin{align} f(b) - f(a) = f'(c)(b-a) \end{align}$$

Rolle's Theorem

If a function $f(x)$ is continous on $[a,b]$ and differentiable on $(a,b)$

And, $f(a) = f(b)$, $ a \neq b$

Then, there is a point $c$ in $(a,b)$ ($c \in (a,b)$)

$$\begin{align} f'(c) = 0 \end{align}$$

What Do Those Theorems Mean?

If you notice in the mean value theorem output, you have the following fraction

$$\begin{align} \frac{f(b) - f(a)}{b-a} \end{align}$$

This looks like the equation for a slope between two points

$$\begin{align} \frac{rise}{run} = \frac{\Delta y}{\Delta x} = \frac{y_2 - y_1}{x_2 - x_1} = \frac{f(b) - f(a)}{b-a} \end{align}$$

What this tells us, is that there is a point within $(a,b)$

Where the tangent line of that point is parallel to the secant line of $(a, f(a))$ and $(b,f(b))$

Take a look at the graph

The green point is in between the two blue points

The tangent line of the green point is parallel to the secant line of the two points

Since they are parallel, they share the same slope


Moving onto Rolle's Theorem

What Rolle's Theorem means is that if two points with different inputs have the same output

There must be an extrema or the function remains constant

This can be thought intutively as if they are the same output with differnet inputs

The function either stays the same, which means no rate of change so, $f'(c) = 0$

Or the function changes value and returns back

It could either increase or decrease, if so it has to do the opposite rate of change to move back to the original output

In that change from increase to decrease or decrease to increase, there has to be a point where it stops either, so either a maxima or minima

A min or max means an extrema, which means a critical point, which means $f'(c) = 0$

Take a look at the graph

Both points are on a horizontal line with same output

The function starts increasing from the first point and then goes decreasing to get back to the second point

The point where the function stops increasing and goes to decreasing is an extrema, the rate of change there is 0

Therefore, $f'(c) = 0$


It's important to know that the Mean Value Theorem and Rolle's Theorem is about the existence of the point

They don't give ways to solve it, it just means you can solve it

Cauchy's Mean Value Theorem

Cauchy's mean value theorem is a more generalized version of the mean value theorem

If the functions $f(x)$ and $g(x)$ are continous on $[a,b]$ and differentiable on $(a,b)$

Then there exists a $c \in (a,b)$ such that

$$\begin{align} (f(b) - f(a))g'(c) = (g(b) - g(a))f'(c) \end{align}$$

This can simplified if $g(a) \neq g(b)$ and $g'(c) \neq 0$

$$\begin{align} \frac{f(b) - f(a)}{g(b) - g(a)} = \frac{f'(c)}{g'(c)} \end{align}$$

  • Find the point $c$ that works for Rolle's Theorem for $f(x) = sin(\pi x)$ on $[1,2]$

    First, verify conditions

    $f(x)$ is differentiable everywhere, therefore continous everywhere

    $$\begin{align} sin(\pi) &= 0 \\ sin(2\pi) &= 0 \\ \end{align}$$

    So we are allowed to use Rolle's Theorem

    $$\begin{align} f(x) = sin(\pi x) \\ f'(x) = \pi cos(\pi x) \end{align}$$

    Solving for 0, for Rolle's Theorem

    $$\begin{align} \pi cos(\pi x) &= 0 \\ cos(\pi x) &= 0 \\ cos^{-1}(cos(\pi x)) &= cos^{-1}(0) \\ \pi x &= \frac{\pi}{2} \\ x &= \frac{1}{2} \end{align}$$

  • Find the point $c$ that works for the Mean Value Theorem for $f(x) = x^3 - 2x^2 - 2$ on $[0,3]$

    First, verify conditions

    $f(x)$ is a polynomial, so it's differentiable everywhere, therefore continous everywhere

    So we are allowed to use the mean value theorem

    $$\begin{align} f'(c) = \frac{f(3) - f(0)}{7 - 2} = \frac{-2 - 7}{5} = -\frac{9}{5} \end{align}$$

  • If $f(x)$ is continous and differentiable everywhere and $f(1) = 3$ and $f'(x) \leq 2$, what's the largest possible value for $f(-3)$?

    Since the function is continous and differentiable everywhere, we can use the Mean Value Theorem

    If we plug our known information into the the mean value theorem,

    $$\begin{align} f'(x) &= \frac{f(1) - f(-1)}{1 - (-1)} \\ f'(x) &= \frac{3 - f(-1)}{2} \\ 2f'(x) &= 3 - f(-1)\\ 2f'(x) -3 &= -f(-1) \\ f(-1) &= -2f'(x) + 3 \\ \end{align}$$

    Since the max value of $f'(x)$ is 2, and we are trying to find the max possible value for $f(0)$

    We plug the max value of $f'(x)$ in

    $$\begin{align} f(-1) &= -2(2) + 3 \\ f(-1) &= -1 \end{align}$$

    So the largest possible value of $f(-1)$ is $-1$

    $$\begin{align} f(-1) \leq -1 \end{align}$$

  • Prove that $f(x) = x^3 - x^2 + 2x + 2$ has only 1 real root using Rolle's Theorem

    $f(x)$ is a polynomial, so it's differentiable everywhere, therefore continous everywhere

    We can use this with the Rolle's, but we need a range

    We need to find a range where the only real root of $f(x)$ could be, we are going to use the Intermediate Value theorem

    $$\begin{align} 0 < &x < -1 \\ f(0) < f(&x) < f(-1) \\ 2 < f(&x) < -2 \end{align}$$

    By The Intermediate Value Theorem/Bolzano's Theorem there is a root between $[-1,0]$

    The interval itself isn't important, but it is important to know there is a root in there


    From here, we are going to do a proof by contradiction

    Assume there are two roots $a$ and $b$, so $f(a) = f(b) = 0$

    So using Rolle's Theorem, there is a $c$ between $a$ and $b$, such that $f'(c) = 0$

    Since $c$ is between $a$ and $b$, $a$ and $b$ are real, therefore $c$ has to be real

    We have to solve for $c$ and show it is not real

    $$\begin{align} f'(x) &= 3x^2 - 2x + 2 \\ x = \frac{2 \pm \sqrt{4 - 24}}{6} &&\text{Using Quadratic Formula} \\ x = \frac{2 \pm \sqrt{-20}}{6} \end{align}$$

    We can see there is no real answer because the square root has a negative inside it

    The first part we proved there is atleast 1 root, the second part we proved there can't be two or more roots

    Putting these both together we get there is only 1 root

Proofs
  • Proof of Mean Value Theorem

    First create a function $g(x)$

    $$\begin{align} g(x) &= f(x) - \frac{f(b) - f(a)}{b-a}x \\ g'(x) &= f'(x) - \frac{f(b) - f(a)}{b-a} \end{align}$$

    With this function $g(x)$ There are two cases

    Case one: The min AND max are at $a$ and $b$

    This means that $g(x)$ is constant, so $g'(x) = 0$ at every point

    $$\begin{align} 0 &= f'(x) - \frac{f(b) - f(a)}{b-a} \\ f'(x) &= \frac{f(b) - f(a)}{b-a} \end{align}$$


    Case two: the extrema is inside $(a,b)$

    If there is an extrema, then there is a point $g'(c) = 0$

    $$\begin{align} 0 &= f'(c) - \frac{f(b) - f(a)}{b-a} \\ f'(c) &= \frac{f(b) - f(a)}{b-a} \end{align}$$

  • Proof of Rolle's Theorem

    Rolle's Theorem is actually a special case of the Mean Value Theorem

    Since:

    $$\begin{align} f(a) = f(b) \rightarrow f(b)-f(a) = 0 \end{align}$$

    Plugging this into the Mean Value theorem

    $$\begin{align} f'(c) = \frac{f(b) - f(a)}{b-a} = \frac{0}{b-a} = 0 \end{align}$$

  • Proof of Cauchy's Mean Value Theorem

    Case one: $g(a) = g(b)$

    $$\begin{align} g(a) = g(b) \rightarrow g(b) - g(a) = 0 \end{align}$$

    $$\begin{align} (f(b) - f(a))g'(c) &= 0 \cdot f'(c) \\ (f(b) - f(a))g'(c) &= 0 \\ g'(c) &= 0 \\ \end{align}$$

    Since $g(a) = g(b)$ and $g'(x)$ exists we can use Rolle's theorem to show there is a $c$ where $g'(c) = 0$

    Case two: $g(a) \neq g(b)$

    Create a function $h(x)$

    $$\begin{align} h(x) = f(x) - \frac{f(b) - f(a)}{g(b) - g(a)}g(x) \end{align}$$

    We see that $h(a) = h(b)$

    $$\begin{align} h(a) - h(b) &= \left[f(a) - \frac{f(b) - f(a)}{g(b) - g(a)}g(a)\right] - \left[f(b) - \frac{f(b) - f(a)}{g(b) - g(a)}g(b)\right] \\ &= f(a) - \frac{f(b) - f(a)}{g(b) - g(a)}g(a) - f(b) + \frac{f(b) - f(a)}{g(b) - g(a)}g(b) \\ &= \frac{f(b) - f(a)}{g(b) - g(a)}(g(b) - g(a)) + f(a) - f(b) \\ &= f(b) - f(a) + f(a) - f(b) \\ &= 0 \end{align}$$

    Since $h(x)$ is continuous on $[a,b]$ and differentiable $(a,b)$ (because $f(x)$ and $g(x)$ are continuous and differentiable)

    We can use Rolle's theorem to find where

    $$\begin{align} h'(c) = 0 \end{align}$$

    $$\begin{align} h'(c) = f'(x) - \frac{f(b) - f(a)}{g(b) - g(a)}g'(x) \\ f'(x) - \frac{f(b) - f(a)}{g(b) - g(a)}g'(x) = 0 \\ f'(x) = \frac{f(b) - f(a)}{g(b) - g(a)}g'(x) \\ \frac{f'(x)}{g'(x)} = \frac{f(b) - f(a)}{g(b) - g(a)} \end{align}$$
