One of the biggest uses of calculus is using derivatives to find extremas (mins and maxes)

The way we achieve this is using critical points

An $x = c$ is a critical point on $f(x)$ if

  1. $f(c)$ exists
  2. $f'(c) = 0$ OR $f'(c)$ is undefined

If $x = c$ is a critical point of $f(x)$, then $f(c)$ is called the critical value

An important often forgotten point is that $f(c)$ HAS TO EXIST

The reason why critical points are related to extrema are to think about the cases of requirement 2

If $f'(c) = 0$ that means the slope/rate of change is 0, which means it stopped changing which could be an extrema

If $f'(c)$ is undefined that means sharp corners or vertical slope, in the case of sharp corners these could also be extrema

It's important to note that every extrema is a critical value BUT every critical value may not be an extrema

Finding Extremas

In order to find extremas, we need to find the critical points, and then use the first derivative test and/or second derivative test


First Derivative Test:

Find a value of $f'(x)$ on the left and right of the critical point $x = c$

Make sure the points on the left and right are not past another critical point

We will call $f'(x)$ on the left $f'(x_{\text{left}})$

We will call $f'(x)$ on the right $f'(x_{\text{right}})$

if $f'(x_{\text{left}}) < 0$ and $f'(x_{\text{right}}) > 0$, then there is a minimum at $c$

if $f'(x_{\text{left}}) > 0$ and $f'(x_{\text{right}}) < 0$, then there is a maximum at $c$

if $f'(x_{\text{left}})$ and $f'(x_{\text{right}})$ are the same sign, there is no extrema


The idea of the first derivative test is that since extrema are the points where the function goes from increasing to decreasing or from decreasing to increasing

since the sign of $f'(x)$ says whether the function is increasing or decreasing

We find if the function is increasing/decreasing on either side to see whether we have a minimum or maximum


Second Derivative Test:

If $f''(c) < 0$, then there is a maximum

If $f''(c) > 0$, then there is a minimum

If $f''(c) = 0$ or $f''(c)$ is undefined, then the test is inconclusvie


The idea here is that $f''(c)$ is the concavity at the point

If concanve up, that means the rate of change is initallliy negative and then keeps increa\sing to positive

Between that change from negative to positive, there has to be a minimum there

Vice-versa for concave down

  • Find the relative extrema of $f(x) = x^5 + 5 x^4 - 5 x^3 - 25 x^2 + 40 x - 16$

    First lets find critical points

    $$\begin{align} f(x) &= x^5 + 5 x^4 - 5 x^3 - 25 x^2 + 40 x - 16 \\ f'(x) &= 5x^4 + 20x^3 - 15x^2 - 50x + 40 \\ \end{align}$$

    $$\begin{align} 5x^4 + 20x^3 - 15x^2 - 50x + 40 &= 0 \\ x^4 + 4x^3 -3x^2 - 10x + 8 &= 0 \end{align}$$

    Using rational root theorem we can factor it

    $$\begin{align} x^4 + 4x^3 -3x^2 - 10x + 8 &= 0 \\ (x-1)^2(x+2)(x+4) &= 0 \\ x = 1, -2, -4 \end{align}$$

    Out of these critical points, let's find which one of these is an extrema u\sing both tests


    First derivative test:

    For $x = 1$, the left point can be $0$ and right is $2$

    $$\begin{align} f'(0) &= 8 \\ f'(1) &= 0 \\ f'(2) &= 24 \\ \end{align}$$

    since $f'(0)$ and $f'(2)$ are both positive (same sign), it is not an extrema

    For $x = -2$, the left point can be $-3$ and right is $0$

    $$\begin{align} f'(-3) &= -16 \\ f'(-2) &= 0 \\ f'(0) &= 8 \\ \end{align}$$

    \since $f'(-3)$ (left point) is negative and $f'(0)$ (right point) is positive, it is a minimum

    For $x = -4$, the left point can be $-5$ and right is $-3$

    $$\begin{align} f'(-5) &= 108 \\ f'(-4) &= 0 \\ f'(-3) &= -16 \\ \end{align}$$

    \since $f'(-5)$ (left point) is positive and $f'(-3)$ (right point) is negative, it is a maximum


    Second deriviative test:

    $$\begin{align} f'(x) &= 5x^4 + 20x^3 - 15x^2 - 50x + 40 \\ f''(x) &= 20x^3 + 60x^2 - 30x - 50 \\ \end{align}$$

    Now let's plug in the critical points

    $$\begin{align} f''(-4) &= -250 \\ f''(-2) &= 90 \\ f''(1) &= 0 \\ \end{align}$$

    Since $f''(4)$ is negative, it is a maximum

    Since $f''(-2)$ is positive, it is a minmum

    Since $f''(1)$ is 0, it is inconclusvie

    Find the relative extrema of $f(x) = \sin(x) + \cos^2(x)$

    First, let's find critical points

    $$\begin{align} f(x) &= \sin(x) + \cos^2(x) \\ f'(x) &= \cos(x) - 2\sin(x)\cos(x) \\ \end{align}$$

    $$\begin{align} \cos(x) - 2\sin(x)\cos(x) &= 0 \\ \cos(x)(1 - 2\sin(x)) &= 0 \\ \cos(x) &= 0 \\ 1-2\sin(x) = 0 \rightarrow \sin(x) &= \frac{1}{2}\\ \end{align}$$

    $$\begin{align} \cos(x) &= 0 \rightarrow x = \frac{\pi}{2} + \pi n, n \in \mathbb{Z}\\ \sin(x) &= \frac{1}{2} \rightarrow x = \frac{\pi}{6} + 2n\pi,\frac{5\pi}{6} + 2n\pi, n \in \mathbb{Z} \\ \end{align}$$

    Combining these two solutions together,

    $$\begin{align} \cos(x) - 2\sin(x)\cos(x) = 0\\ x = \frac{\pi}{2} + \pi n, \frac{\pi}{6} + 2n\pi, \frac{5\pi}{6} + 2n\pi,n \in \mathbb{Z} \\ \end{align}$$

    Using the first derivative test would probably be complicated, so I'm going to use the second derivative test

    $$\begin{align} f'(x) &= \cos(x) - 2\sin(x)\cos(x) \\ f''(x) &= -\sin(x) + 2(\sin^2(x) - \cos^2(x)) \\ &= -\sin(x) - 2(\cos^2(x) - \sin^2(x)) \\ &= -\sin(x) - 2\cos(2x) &&\text{U\sing \co\sine Double Angle Property} \\ &= -\sin(x) - 2(1 - 2\sin^2(x)) &&\text{U\sing \co\sine Double Angle Property} \\ &= -\sin(x) - 2 + 4\sin^2(x) \\ &= 4\sin^2(x) - \sin(x) - 2 \end{align}$$

    What you can notice is that $f''(x)$ is in the form of a quadratic (where $x$ is $\sin(x)$)

    We can use the quadratic formula to find where the x-values of the roots are

    $$\begin{align} \sin(x) = \frac{1 \pm \sqrt{(-1)^2 - 4(4)(-2)}}{2(4)} = \frac{1 \pm \sqrt{33}}{8} + 2n\pi \approx 0.84307 + 2n\pi, -.59307 + 2n\pi \end{align}$$

    And u\sing vertex formula $\frac{-b}{2a}$, we can find the vertex to see if it's positive or negative

    $$\begin{align} x = \frac{1}{8} f''(\frac{1}{8}) \approx -2.0624 \end{align}$$

    The reason why we did all this is to show if our critical point is between is the roots, then it will be negative

    If it's outside the roots then it will be positive

    This is because the vertex lies in the middle of the roots and it is negative

    Adding $n \pi$ or $2n\pi$ won't matter because the roots are all being added by $2n\pi$

    $$\begin{align} f''(\frac{\pi}{2} + \pi n) > 0 \\ f''(\frac{\pi}{6} + 2n\pi) < 0 \\ f''(\frac{5\pi}{6} + 2n\pi) < 0 \\ \end{align}$$

    This means that,
