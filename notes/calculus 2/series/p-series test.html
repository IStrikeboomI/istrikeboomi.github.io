A P-Series is a series in the form:

$$\begin{align} \sum_{n=1}^\infty{\frac{1}{n^p}} \end{align}$$

Where $p$ is a number

A P-Series converges if $p > 1$

A P-Series diverges if $p \leq 1$


P-series (along with geometric series) are the easiest tests to do

Because of this, they serve as a starting basis for choosing a series to compare to

Proof

The P-Series test is a special case of the Integral test

$$\begin{align} \int_1^\infty{\frac{1}{n^p} \ dn} = \lim_{t \to \infty}\int_1^t{\frac{1}{n^p} \ dn} \end{align}$$

If $p = 1$

$$\begin{align} \int_1^\infty{\frac{1}{n} \ dn} &= \lim_{t \to \infty}\int_1^t{\frac{1}{n} \ dn} \\ &= \lim_{t \to \infty}\left[\ln(|n|)\right]_1^t\\ &= \infty\\ \end{align}$$

So it diverges if $p = 1$

If $p \neq 1$

We can solve this using the power rule if $p \neq 1$

$$\begin{align} \lim_{t \to \infty}\int_1^t{\frac{1}{n^p} \ dn} &= \lim_{t \to \infty}\left[\frac{n^{-p + 1}}{-p + 1}\right]_1^t \\ &= \lim_{t \to \infty}\left[\frac{n^{-(p - 1)}}{-p + 1}\right]_1^t \\ &= \lim_{t \to \infty}\left[\frac{1}{n^{p - 1}(-p + 1)}\right]_1^t \\ &= \lim_{t \to \infty}\frac{1}{t^{p - 1}(-p + 1)} - \frac{1}{-p + 1} \\ &= \lim_{t \to \infty}\frac{t^0}{t^{p - 1}(-p + 1)} - \frac{1}{-p + 1} \\ \end{align}$$

If you remember Limits at infinity, we get 0 if degrees on the bottom is larger than top

So we get the inequality, if $p-1 > 0 \rightarrow p > 1$, then the integral converges

Otherwise, if $p < 1$, the integral diverges

Practice Problems
  • Determine Convergence Or Divergence of $\displaystyle\sum_{n = 1}^\infty {\frac{1}{n}}$

    $$\begin{align} \displaystyle\sum_{n = 1}^\infty {\frac{1}{n}} \end{align}$$

    $p = 1$, so it diverges

  • Determine Convergence Or Divergence of $\displaystyle\sum_{n = 1}^\infty {\frac{1}{n\sqrt{n}}}$

    $$\begin{align} \displaystyle\sum_{n = 1}^\infty {\frac{1}{n\sqrt{n}}} = \displaystyle\sum_{n = 1}^\infty {\frac{1}{n^{3/2}}} \end{align}$$

    $p = \frac{3}{2}$, so it converges

  • Determine Convergence Or Divergence of $\displaystyle\sum_{n = 1}^\infty {\frac{1}{n^2}}$

    $$\begin{align} \displaystyle\sum_{n = 1}^\infty {\frac{1}{n^2}} \end{align}$$

    $p = 2$, so it converges
