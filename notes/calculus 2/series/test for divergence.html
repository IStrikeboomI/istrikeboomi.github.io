The Test For Divergence (TFD) says for a series:

$$\begin{align} \sum_{n = k}^\infty{a_n} \end{align}$$

If:

$$\begin{align} \lim_{n \to \infty}{a_n} \neq 0 \end{align}$$

Then the series diverges

IMPORTANT: if the limit does equal zero, the test is inconclusive

This means the series could diverge or it could converge

This is one the most basic divergence tests

It should be used as one of the first tests if the limit "looks" like it won't go to $0$


The reasoning behind this test is that if the limit isn't zero at infinity,

That means it will be adding a finite amount of terms

Example:

$$\begin{align} \sum_{n = k}^\infty{\frac{3n^2}{2n^2 + n}} = \frac{3}{5} + \frac{12}{10} + \frac{27}{21} + \ldots + \frac{3}{2} + \frac{3}{2} + \ldots = \infty \end{align}$$

$$\begin{align} \lim_{n \to \infty}\frac{3n^2}{2n^2 + n} = \frac{3}{2} \neq 0 \end{align}$$

So this series diverges

Practice Problems
  • Use the TFD on $\displaystyle\sum_{n = 4}^\infty{\frac{e^x}{2^x}}$

    $$\begin{align} \sum_{n = 4}^\infty{\frac{e^x}{2^x}} \end{align}$$

    $$\begin{align} \lim_{n \to \infty}{\frac{e^x}{2^x}} = \infty \end{align}$$

    So this series diverges

  • Use the TFD on $\displaystyle\sum_{n = 1}^\infty{\frac{\sqrt{x}}{x^2}}$

    $$\begin{align} \sum_{n = 1}^\infty{\frac{\sqrt{x}}{x^2}} \end{align}$$

    $$\begin{align} \lim_{n \to \infty}{\frac{\sqrt{x}}{x^2}} = 0 \end{align}$$

    So the TFD is inconclusive so this series could diverge or converge

  • Use the TFD on $\displaystyle\sum_{n = 1}^\infty{n}$

    $$\begin{align} \sum_{n = 1}^\infty{n} \end{align}$$

    $$\begin{align} \lim_{n \to \infty}{n} = \infty \end{align}$$

    So this series diverges

  • Use the TFD on $\displaystyle\sum_{n = 1}^\infty{(-1)^n}$

    $$\begin{align} \sum_{n = 1}^\infty{(-1)^n} \end{align}$$

    $$\begin{align} \lim_{n \to \infty}{(-1)^n} = DNE \end{align}$$

    So this series diverges
