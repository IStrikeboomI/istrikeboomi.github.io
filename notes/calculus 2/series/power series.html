A Power Series centered at $a$ is a series in the form

$$\begin{align} \sum_{n = 0}^{\infty}{c_n(x - a)^n} = c_0 + c_1(x-a) + c_2(x-a)^2 + \ldots \end{align}$$

Where $c_n$ and $k$ are real numbers

A power series can be used to represent a function as an infinite series

The center of the power series is where the approximation (and nearby points) is closest

Power Series are also useful because they appear like polynomials


Convergence

A Power Series has one of 3 possibilities of convergence

  1. A Power Series converges only at $x=a$, $R=0$
  2. A Power Series converges for all $x$, $R=\infty$
  3. A Power Series converges in range $R$ from $x$ on either side, $|x-a| < R$

This $R$ value is refered to as the Radius Of Convergence

The Interval Of Convergence is the interval of the values that make a series converge

To find the interval of converge $I = (x-a,x+a)$, but make sure to check the endpoints


For example:

The power series for $\ln(1+x)$ (centered at 0) is:

$$\begin{align} \ln(1+x) = \sum_{n = 1}^\infty{\frac{(-1)^{n-1} x^n}{n}} \end{align}$$

This particular series has an $R=1$ and an interval of convergence of $I = (-1,1]$

If we were to center this series $\ln(x)$ at $x=1$

$$\begin{align} \ln(x) = \sum_{n = 1}^\infty{\frac{(-1)^{n-1} (x-1)^n}{n}} \end{align}$$

This series has the same $R=1$ and an interval of convergence of $I = (-1,1]$

If we graph the power series to a few terms vs the graph of $\ln(x+1)$

We see the series is very close to the center point $x=0$ and surroudning points

After $x > 1$, you can see how the series diverges

Deriving Power Series

Remember the geometric series formula

$$\begin{align} \sum_{n = 0}^\infty{x^n} = 1 + x + x^2 + x^3 + \ldots = \frac{1}{1-x} &&\text{If $|x| < 1$} \end{align}$$

Also, if a power series has an $R > 0$

And the function it represents is differentiable and/or integratable on $(x-a,x+a)$

$$\begin{align} f(x) = \sum_{n=0}^\infty{c_n(x-a)^n} \end{align}$$

Then,

$$\begin{align} f'(x) = \sum_{n=1}^\infty{nc_n(x-a)^{n-1}} \\ \int{f(x) \ dx} = C+\sum_{n=1}^\infty{c_n\frac{(x-a)^{n+1}}{n+1}} \end{align}$$

Have the same radius of convergence, but the endpoints might not be the same

Notice how the the derivative series starts at $1$ and not $0$

This is because the first term is constant so it goes away

And also notice how the constant of integration is present so you need to solve for it

Finding The Radius And Interval Of Convergence

To find $R$, we need to make the inequality $|x-a| < R$

So which convergence tests have an inequality?

  • Ratio Test
  • Root Test
  • Geometric Series Test
  • P-Series Test

But most of the time you will be using the Ratio Test or the Root Test

In the ratio and root test, the limit has $n$ going to $\infty$

But the Inequality for $R$ has $x$, this means using Limit Properties

You can then isolate $|x|$ and factor it out

Example

Find the radius and interval of convergence of the following series

$$\begin{align} \sum_{n = 0}^\infty{\frac{x^n}{n!}} \end{align}$$

I will use the Ratio Test because of the factorial and it gives an inequality

$$\begin{align} \lim_{n \to \infty}{\left|\frac{x^{n+1}}{(n+1)!} \cdot \frac{n!}{x^{n}}\right|} = \lim_{n \to \infty}{\left|\frac{x}{n+1}\right|} = |x| \cdot \lim_{n \to \infty}{\left|\frac{1}{n+1}\right|} = |x| \cdot 0 = 0 \\ 0 < 1 \end{align}$$

This shows the radius of convergence is infinity, so this series works everywhere

Since $R = \infty$, there are no endpoints we need to test for


Find the power series for $f(x) = \arctan(x)$ and then find the radius and inteveral of convergence

$$\begin{align} \arctan(x) &= \int{\frac{1}{1 + x^2} \ dx} + C \\ \frac{1}{1 + x^2} &= \frac{1}{1-(-x^2)} = \sum_{n = 0}^\infty{(-x^2)^n} = \sum_{n = 0}^\infty{(-1)^nx^{2n}} &&\text{If $|-x^2| < 1 \rightarrow |x| < 1$} \\ \end{align}$$

$$\begin{align} \int{\frac{1}{1 + x^2} \ dx} &= \int{\sum_{n = 0}^\infty{(-1)^nx^{2n}} \ dx} = \sum_{n = 0}^\infty{\left((-1)^n\int{x^{2n} \ dx}\right)} = C+\sum_{n = 0}^\infty{(-1)^n\frac{x^{2n + 1}}{2n + 1}} \\ \arctan(x) &= C+\sum_{n = 0}^\infty{(-1)^n\frac{x^{2n + 1}}{2n + 1}} \\ \end{align}$$

So we almost have a series for $\arctan(x)$, but we need to find what $C$ is

To do this I am going to plug in $x = 0$ to make the series go to $0$

$$\begin{align} \arctan(0) &= C+\sum_{n = 0}^\infty{(-1)^n\frac{0^{2n + 1}}{2n + 1}} \\ 0 &= C + 0 \\ C &= 0 \end{align}$$

We can plug this $C$ back into the above equation

$$\begin{align} \arctan(x) &= \sum_{n = 0}^\infty{(-1)^n\frac{x^{2n + 1}}{2n + 1}} \\ \end{align}$$

Now we have an infinite polynomial for $\arctan(x)$!, but we still need to find $R$

Since the geometric series we used has $R = 1$, this series will also have $R = 1$

To find the interval we need the test the endpoints of $-1$ and $1$

$x = 1$

$$\begin{align} \sum_{n = 0}^\infty{(-1)^n\frac{1^{2n + 1}}{2n + 1}} = \sum_{n = 0}^\infty{(-1)^n\frac{1}{2n + 1}} \\ \end{align}$$

This series converges by the Alternating Series Test

$x = -1$

$$\begin{align} \sum_{n = 0}^\infty{(-1)^n\frac{(-1)^{2n + 1}}{2n + 1}} = \sum_{n = 0}^\infty{(-1)^n\frac{-1}{2n + 1}} = -\sum_{n = 0}^\infty{(-1)^n\frac{1}{2n + 1}}\\ \end{align}$$

This series converges by the Alternating Series Test

So both endpoints converge so this series has an interval of convergence of $I = [-1,1]$

Practice Problems
  • Find the radius and interval of convergence of $\displaystyle\sum_{n = 1}^\infty{\frac{(3x - 1)^n}{2^n\sqrt{n+1}}}$

    Using the ratio test

    $$\begin{align} \lim_{n \to \infty}{\left|\frac{(3x - 1)^{n+1}}{2^{n+1}\sqrt{n+2}} \cdot \frac{2^n\sqrt{n+1}}{(3x - 1)^n}\right|} &= \lim_{n \to \infty}{\left|\frac{(3x - 1)\sqrt{n+1}}{2\sqrt{n+2}}\right|} \\ &= \left|\frac{3x-1}{2}\right|\lim_{n \to \infty}{\frac{\sqrt{n+1}}{\sqrt{n+2}}} \\ &= \left|\frac{3x-1}{2}\right|\lim_{n \to \infty}{\sqrt{\frac{n+1}{n+2}}} \\ &= \left|\frac{3x-1}{2}\right|\sqrt{\lim_{n \to \infty}{{\frac{n+1}{n+2}}}}&&\text{Because $\sqrt{n}$ is Continous}\\ &= \left|\frac{3x-1}{2}\right|\cdot 1\\ \left|\frac{3x-1}{2}\right| &< 1 \\ -1 < &\frac{3x-1}{2} < 1 \\ -2 < &3x-1 < 2 \\ -1 < &3x < 3 \\ -1/3 < &x < 1 \\ \end{align}$$

    To get $R$, we need to rewrite the inequality

    $$\begin{align} \left|\frac{3x-1}{2}\right| &< 1 \\ \left|3x-1\right| &< 2 \\ \left|3\left(x-\frac{1}{3}\right)\right| &< 2 \\ \left|x-\frac{1}{3}\right| &< \frac{2}{3} \\ \end{align}$$

    So we get $R = \frac{2}{3}$, to find the interval we use the endpoints above

    At $x = -1/3$

    $$\begin{align} \sum_{n = 1}^\infty{\frac{(3(1/3) - 1)^n}{2^n\sqrt{n+1}}} = \sum_{n = 1}^\infty{\frac{0^n}{2^n\sqrt{n+1}}} = 0 \end{align}$$

    So it converges

    At $x = 1$

    $$\begin{align} \sum_{n = 1}^\infty{\frac{(3(1) - 1)^n}{2^n\sqrt{n+1}}} = \sum_{n = 1}^\infty{\frac{2^n}{2^n\sqrt{n+1}}} = \sum_{n = 1}^\infty{\frac{1}{\sqrt{n+1}}} = \infty \end{align}$$

    So this series diverges by the Limit Comparison Test combined with the P-series test using the series $\displaystyle\sum_{n=1}^\infty{\frac{1}{\sqrt{n}}}$

    This finally gives us an $R = \frac{2}{3}$ and $I=[\frac{1}{3},1)$

  • Find a power series for $\frac{1}{(1-x)^2}$ and it's radius and interval of convergence

    $$\begin{align} \frac{1}{(1-x)^2} &= \frac{d}{dx}\left(\frac{1}{1-x}\right) \\ &= \frac{d}{dx}\left(\sum_{n = 0}^\infty{x^n}\right) &&\text{If $|x|<1 $}\\ &= \sum_{n = 1}^\infty{nx^{n-1}} \\ \end{align}$$

    So here $R = 1$, now to test the endpoints

    $x = 1$

    $$\begin{align} \sum_{n = 1}^\infty{n1^{n-1}} \\ \end{align}$$

    Diverges by Test For Divergence

    $x = -1$

    $$\begin{align} \sum_{n = 1}^\infty{n(-1)^{n-1}} \\ \end{align}$$

    Diverges by Test For Divergence

    So $I = (-1,1)$
