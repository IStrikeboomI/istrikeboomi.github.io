A Telescoping Series is a series where part of the terms are cancelled by future terms

For Example:

$$\begin{align} \sum_{n = 1}^k{a_n - a_{n+1}} &= (a_1 - a_2) + (a_2 - a_3) + (a_3 - a_4) + \ldots + (a_{k-1} - a_k) + (a_k - a_{k+1})\\ &= a_1 - a_{k+1} \end{align}$$

If we make this into an infinite series (Assuming $\lim_{n \to \infty}a_n = 0$)

$$\begin{align} \sum_{n = 1}^\infty{a_n - a_{n+1}} &= \lim_{n \to \infty}{a_1 - a_{n+1}} = a_1 \end{align}$$

This is called a telescoping series because it opens up and expands like a telescope

And then cancelling the terms collapses the telescope into a few terms

Examples

$$\begin{align} \sum_{n = 1}^\infty{\left(\frac{1}{n} - \frac{1}{n+1}\right)} &= \left(\frac{1}{1} - \frac{1}{2}\right) + \left(\frac{1}{2} - \frac{1}{3}\right) + \ldots + \left(\frac{1}{n-1} - \frac{1}{n}\right) + \left(\frac{1}{n} - \frac{1}{n+1}\right) \\ &= 1 - \frac{1}{n + 1} \end{align}$$

Here the second term is cancelled by the first term in the next iteration

So whats left is the absolute first term (1) and the last term ($1/n+1$)

Since this is the partial sum $S_n$ if we take the limit to infinity we get the actual sum

$$\begin{align} S_n &= 1 - \frac{1}{n + 1} \\ \lim_{n \to \infty}S_n &= \lim_{n \to \infty}1 - \frac{1}{n + 1} = 1 \\ \end{align}$$

So this series equals $1$

$$\begin{align} \sum_{n = 1}^\infty{\left(\frac{1}{n} - \frac{1}{n+1}\right)} &= 1 \end{align}$$


$$\begin{align} \sum_{n = 2}^\infty{\ln\left(\frac{n+1}{n-1}\right)} = \sum_{n = 2}^\infty{\ln(n+1) - \ln(n-1)} &= (\ln(3) - \ln(1)) + (\ln(4) - \ln(2)) + (\ln(5) - \ln(3)) + \ldots \\ &+ (\ln(n-1) - \ln(n-3)) + (\ln(n) - \ln(n-2)) + (\ln(n+1) - \ln(n-1)) \\ &= -\ln(1) - \ln(2) + \ln(n) + \ln(n-1) \end{align}$$

Now taking the limit to infinity

$$\begin{align} S_n &= -\ln(1) - \ln(2) + \ln(n) + \ln(n-1) \\ \lim_{n \to \infty}S_n &= \lim_{n \to \infty}-\ln(1) - \ln(2) + \ln(n) + \ln(n-1) \\ &= \infty \end{align}$$

So this series diverges

Practice Problems
  • Determine the value of $\displaystyle\sum_{n = 2}^\infty{\frac{1}{n^2 - n}}$

    $$\begin{align} \sum_{n = 2}^\infty{\frac{1}{n^2 - n}} = \sum_{n = 2}^\infty{\frac{1}{n(n-1)}} = \sum_{n = 2}^\infty{\frac{1}{n-1} - \frac{1}{n}} \end{align}$$

    This was obtained using partial fraction decomposition

    $$\begin{align} \sum_{n = 2}^\infty{\frac{1}{n-1} - \frac{1}{n}} &= \left(\frac{1}{1} - \frac{1}{2}\right) + \left(\frac{1}{2} - \frac{1}{3}\right) + \left(\frac{1}{3} - \frac{1}{4}\right) + \ldots \\ &+ \left(\frac{1}{n-3} - \frac{1}{n-2}\right) + \left(\frac{1}{n-2} - \frac{1}{n-1}\right) + \left(\frac{1}{n-1} - \frac{1}{n}\right) \\ &= 1 - \frac{1}{n} \end{align}$$

    Taking the limit

    $$\begin{align} S_n &= 1 - \frac{1}{n}\\ \lim_{n \to \infty}S_n &= \lim_{n \to \infty}1 - \frac{1}{n} = 1\\ \end{align}$$

    So this series converges to $1$

  • Determine the value of $\displaystyle\sum_{n = 1}^\infty{\ln\left(1 + \frac{1}{n}\right)}$

    $$\begin{align} \sum_{n = 1}^\infty{\ln\left(1 + \frac{1}{n}\right)} = \sum_{n = 1}^\infty{\ln\left(\frac{n+1}{n}\right)} = \sum_{n = 1}^\infty{\ln(n+1) - \ln(n)} \end{align}$$

    Expanding the terms

    $$\begin{align} \sum_{n = 1}^\infty{\ln(n+1) - \ln(n)} &= (\ln(2) - \ln(1)) + (\ln(3) - \ln(2)) + (\ln(4) - \ln(3)) + \ldots \\ &+ (\ln(n-1) - \ln(n-2)) + (\ln(n) - \ln(n - 1)) + (\ln(n+1) - \ln(n)) \\ &= -\ln(1) + \ln(n+1) \end{align}$$

    Taking the limit

    $$\begin{align} S_n &= -\ln(1) + \ln(n+1) \\ \lim_{n \to \infty}S_n &= \lim_{n \to \infty}-\ln(1) + \ln(n+1) = \infty\\ \end{align}$$

    So this series diverges
