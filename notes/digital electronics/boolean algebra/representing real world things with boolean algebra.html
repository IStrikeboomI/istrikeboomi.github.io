So far we have learned many boolean algebra and binary concepts, but why is what we learned important?

It's because eventually we will use those principals to model, design, and build circuits

But when we design circuits, we need to know how to convert between real world problems/statments

This involves making variables from our input parameters (real world these can be sensors, switches, etc.)

Then we need to use boolean algebra to represent the logic of the system

Example 1: Simple Security System

Imagine we want to design a simple security system for a house

The security system is an alarm that goes off (output) when it is not daytime and motion is detected

We can represent daytime as a boolean variable `D` (true if daytime, false if night)

We can represent motion detected as a boolean variable `M` (true if motion detected, false if no motion)

Now we can create a boolean expression for when the alarm should go off

$A = D' \cdot M$

We will show later how to convert this into a digital circuit

Example 2: Fan Temperature Cooling System

Imagine a fan that turns on when it is manually turn on or temperature is too high and it has been 30 seconds

Manual switch is $S$, temperature too hot is $T$, and time past 30s is $P$

$F = S + TP$

Example 3: Cat Cafe

Imagine a cat cafe that has a spot for people to go in

There are 2 cats that come in controlled by electronic doors

The cats hate each other, but they like people

The people are happy when the cats are happy, AKA not fighting (so only 1 cat at a time)

Let $C_1$ and $C_2$ be the boolean variables for cat 1 and cat 2 being in the cafe

$H = C_1'C_2 + C_2'C1 = C_1 \oplus C_2$

Convert into a sentence

Let's say we have the boolean expression:

$$G = C'(S + A + W)$$

  • $G$ means the grass is green
  • $C$ means it is cold
  • $S$ means it is summer
  • $A$ means it is spring
  • $W$ means the grass has been watered

So the sentence is:

The grass is green when it is not cold and it is either summer, spring, or the grass has been watered.

Practice Problems
  • It is humid if someone is cooking or taking a hot shower and the dehumidifier is off

    $$H = (C + S)D'$$

  • The TV is off if no one is home or there is an energy watch
    • $T$ means the TV is on
    • $H$ means someone is home
    • $E$ means there is an energy watch

    $T' = H' + E$

    We can use De Morgan's law to simplify this:

    $$T = HE'$$

  • You can only drive if you have gas money and a leaner permit and someone else
    with a license is in the car or if you have a drivers license
    • $M$ means you have gas money
    • $L$ means you have a learner permit
    • $S$ means someone with a license is in the car
    • $D$ means you have a driver's license

    $C = M(LS + D)$
